#  SalesManagement – Retail POS & Supplier Management System

A Java-based console application that simulates a Retail Point-of-Sale (POS) system with supplier and inventory management. This project is designed to demonstrate object-oriented programming principles in a real-world business scenario.

##  Features

- Add and manage products with tax categories
- Add and manage suppliers
- Create and receive purchase orders
- Make sales with stock validation
- Automatic stock updates after purchases and sales
- Tax calculations per product category
- Menu-driven console interface with input validation
- Displays sale bills and purchase receipts with detailed breakdowns

---

##  Package Structure

com.kce.pos.model → Domain classes (Product, Supplier, Sale, etc.) com.kce.pos.service → Business logic for inventory and POS operations com.kce.pos.exception → Custom exception handling com.kce.pos.util → Input validation utilities com.kce.pos.main → Main class with menu-driven interface
